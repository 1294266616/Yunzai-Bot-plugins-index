# 贡献指南

按照如下格式组合：（注意符号都是半角，多个作者请用空格分隔）

```markdown
| [插件名称](插件主页地址) | [@作者名](作者主页地址) | 支持版本 | 备注 |
```

将组合好的内容添加到 `README.md` 中，然后发起 `pull request` 


提交需合理按分类推送共享,若推送有冲突将拒绝处理。

因过多推送造成严重冲突提示,请自觉撤回推送并解决冲突后再`pull request` 

JS小插件不推荐使用[大写/中文].js等特殊字符

推荐命名为为

name-key.js

name_key.js

name.key.js
